<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/SAHU-01/PostIt.git
```

Go to the client directory

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Start the client

```bash
  npm start
```

Go to the server directory

```bash
  cd server
```

Install dependencies

```bash
  npm i sanity
```

Start the client

```bash
 sanity start
```

Check the Databas by

```bash
 sanity manage
```
